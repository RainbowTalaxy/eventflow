## 查看事件流程
GET /event/list

参数名|类型|必填|描述|默认值
:--|:--|:--|:--|:--
t_id    |number |是 |工作组 ID

返回值样例：
```js
{
    "status": {
        "code": 100,
        "message": '成功'
    }
    "list": [
        {
            "f_id": 101, // 事件流程 ID
            "completed": ["肥宅"], // 已完成人员
            "current": "宿霸", // 当前执行事件的人
            "inqueue": ["舍暴", "11"], // 待命的人
            "self": false, // 当前是否为自己（是否可提交“事件完成”申请）
            "isFounder": false, // 是否为工作组组长（是否可删除事件流程）
            "submitted": true // 当前执行事件的人是否提交了“事件完成”申请
        },
        {
            "f_id": 102,
            "completed": ["wjf", "pxx", "jyx", "hc", "cjw", "gsy"],
            "current": "csw",
            "inqueue": ["hdj", "xcj", "gq", "zrl", "yhl"],
            "self": true,
            "isFounder": false,
            "submitted": false
        }
    ]
}
```

## 添加事件流程

:--|:--|:--|:--|:--
t_id    |number |是 |工作组 ID
